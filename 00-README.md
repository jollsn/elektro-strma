# Smart Home + FVE + EPS + Loxone + 2 hlavní rozvaděče

## Stav dokumentu
Pracovní návrh / podklad pro projekt, schéma a kusovník.

## Objekt
Rodinný dům:
- sklep
- přízemí
- patro

Vedlejší objekty:
- garáž
- zahradní bouda

## Hlavní koncepce
- R1: Loxone / 24V / SELV / řízení
- R2: silový / FVE / EPS / jištění
- samostatný datový rack
- podružný rozvaděč garáž
- podružný rozvaděč bouda
- lokální světelná krabice patro

## Rozvaděče
- 2x hlavní rozvaděč
- každý: 6 řad x 24 modulů = 144M

## Přívod
- 3x400/230V
- hlavní jistič: 3x25A
- distributor: ČEZ
- sazba: D45D

## FVE
- měnič: SolaX X3-Hybrid-15.0-D
- FV výkon: 9,96 kWp
- stringy: 2
- baterie: 23,2 kWh
- měření pro SolaX: DTSU666-CT
- EPS: integrované v měniči
- hromosvod: ano

## Důležité provozní rozhodnutí
- hlavní dům na EPS
- garáž mimo EPS
- bouda mimo EPS
- wallbox mimo EPS
- podlahy při blackoutu vypnout
- sauna při blackoutu vypnout
- žebřík při blackoutu vypnout
- RGBW pásky při blackoutu vypnout
